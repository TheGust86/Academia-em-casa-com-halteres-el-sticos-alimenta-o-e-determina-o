# Academia-em-casa-com-halteres-el-sticos-alimenta-o-e-determina-o
Notebook com plano de musculação para homem iniciante de 40 anos com perfil ectomorfo. Engloba um plano completo dividido em três dias: A para A para peito, ombros e tríceps; B para pernas e abdômen; e C para bíceps e antebraço. Também há dicas sobre alimentação, tempo de descanso entre séries e progressão de carga.
